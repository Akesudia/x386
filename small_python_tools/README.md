#### How to use?

	Windows:

	1) open cmd, and type command: arp -a > arp.txt

    2) python oui.py --arp-tab-file ./arp.txt


    Linux:

	1) arp -n > arp_linux.txt

	2) python oui.py --arp-tab-file ./arp_linux.txt


    all output will be saved to result.txt
